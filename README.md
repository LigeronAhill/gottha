# Шаблон для сайта на Go, Templ, Tailwindcss, HTMX, AlpineJS (GOTTHA)

1. Установить [just](https://just.systems/man/en/packages.html)
2. Установить зависимости
	```bash
	just install-deps
	```
3. Посмотреть доступные команды
	```bash
	just help
	```

- [favicon](https://github.com/pixel-point/favpie)
- Добавить Title, Description, OpenGraph (og-title, og-description, og-image) [habr](https://habr.com/ru/companies/click/articles/492258/)
- [sitemap](https://github.com/sabloger/sitemap-generator)
